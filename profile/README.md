## About

**HireDiverSity Play**는 새로운 아이디어를 빠르게 만들고, 검증하고, 배포하기 위한 실험 공간입니다.

이 조직에서는 다음과 같은 작업을 다룹니다.

- 신규 서비스 아이디어 검증
- 랜딩 페이지 및 홍보 페이지 제작
- 미니 서비스 / 인터랙션 페이지 제작
- Netlify 기반 프론트엔드 배포
- Supabase 연동 실험
- QA 자동화 및 테스트 환경 검증
